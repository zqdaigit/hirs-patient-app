# HIRS 患者端 App 开发计划

## 版本规划

### v1.0 (MVP) - 3个月

#### 阶段1：基础框架 (1个月)
- [ ] 项目初始化 (Expo + TypeScript)
- [ ] 用户认证系统
- [ ] 基础 UI 框架
- [ ] 导航配置
- [ ] 状态管理搭建

#### 阶段2：核心功能 (1-2个月)
- [ ] 训练计划模块
- [ ] 训练执行模块
- [ ] 训练记录模块
- [ ] 进度统计模块

#### 阶段3：增强功能 (1个月)
- [ ] 患者教育模块
- [ ] 复诊准备模块
- [ ] 个人设置
- [ ] 测试与优化

## 功能优先级

- [ ] 邀请码注册

- [ ] 执行训练
- [ ] 训练记录
- [ ] 基本统计

### P1 - 1.0版本应该有
- [ ] 知识库
- [ ] 进度图表
- [ ] 报告导出
- [ ] 通知提醒

### P2 - 后续版本
- [ ] 成就系统
- [ ] 社交功能
- [ ] 多语言
