## 1. 项目概述

### 1.1 产品定位

HIRS 患者端 App 是一款面向脊柱侧弯患者的居家康复训练移动应用，配合康复师端和 Web 端使用，实现"院外康复，院内指导"的新模式。

### 1.2 核心功能

| 功能 | 说明 |
|------|------|
| 训练计划 | 查看康复师制定的个性化训练方案 |
| 训练执行 | 视频演示 + 计时引导完成训练 |
| 进度跟踪 | 训练数据统计和趋势分析 |
| 复诊准备 | 生成复诊数据报告 |
| 患者教育 | 脊柱侧弯康复知识库 |

### 1.3 用户角色

- **患者**: 居家康复训练的用户，需要按计划执行训练

## 4. 功能模块

### 4.1 用户认证

#### 4.1.1 邀请码注册

| 字段 | 说明 | 验证规则 |
|------|------|----------|
| 邀请码 | 康复师提供 | 必填，6位字母数字组合 |
| 手机号 | 登录账号 | 必填，11位手机号 |
| 密码 | 登录密码 | 必填，6-20位 |
| 确认密码 | 确认密码 | 必须与密码一致 |

**注册流程**：
1. 用户输入康复师提供的邀请码
2. 系统验证邀请码有效性（有效期7天）
3. 验证通过后，显示注册表单
4. 填写手机号、密码、确认密码
5. 提交注册，成功后自动登录

#### 4.1.2 手机号登录

| 字段 | 说明 | 验证规则 |
|------|------|----------|
| 手机号 | 登录账号 | 必填，11位手机号 |
| 密码 | 登录密码 | 必填 |

**登录流程**：
1. 输入手机号和密码
2. 验证 credentials
3. 返回 JWT token
4. 跳转首页

#### 4.1.3 密码重置

| 步骤 | 说明 |
|------|------|
| 1 | 输入注册时的手机号 |
| 2 | 获取短信验证码（60秒可重发） |
| 3 | 输入新密码 |
| 4 | 确认修改成功 |

#### 功能清单

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 邀请码注册 | 输入康复师提供的邀请码注册 | P0 |
| 手机号登录 | 手机号 + 密码登录 | P0 |
| 完善个人资料 | 填写姓名、性别、出生日期 | P0 |
| 密码重置 | 忘记密码，通过验证码重置 | P1 |
| 修改密码 | 登录后修改密码 | P1 |
| 注销账号 | 申请注销账号（管理员审批） | P2 |

```

│ ← 返回    训练中    暂停      │
├────────────────────────────────┤
│                                │
│     ┌─────────────────┐       │

│        ════════               │

│  ┌─────┐ ┌─────┐ ┌─────┐     │

│  └─────┘ └─────┘ └─────┘     │
```

#### 4.4.2 动作演示

| 演示形式 | 说明 |
|----------|------|
| 视频播放 | MP4 格式，自动循环播放 |

| 静态图片 | 动作要领截图 |

#### 4.4.3 训练计时器

| 状态 | 显示 |
|------|------|
| 倒计时中 | 大字体数字，如 "00:25" |
| 组间休息 | 休息倒计时，显示"休息中" |
| 暂停 | 保持当前时间，显示"已暂停" |

#### 4.4.4 实时反馈

| 按钮 | 含义 | 记录 |
|------|------|------|
| ✅ 完成 | 动作顺利完成 | completed |
| ⚠️ 注意 | 动作有偏差需注意 | attention |
| 🔄 调整 | 需要调整姿势 | adjusted |
| 🔴 疼痛 | 感到疼痛 | pain |

#### 4.4.5 训练控制

| 操作 | 说明 |

| 继续 | 恢复暂停的计时 |
| 上一个 | 跳转到前一动作 |
| 下一个 | 跳过当前动作 |
| 提前结束 | 结束本次训练 |
