# HIRS 患者端 App

> HIRS (Home-based Intelligent Rehabilitation System for Scoliosis)
> 脊柱侧弯智能康复辅助系统 - 患者端移动应用

## 项目简介

HIRS 患者端 App 是面向脊柱侧弯患者的居家康复训练应用，提供训练计划执行、进度跟踪、数据上传等功能。

## 技术栈

- **框架**: React Native + Expo
- **语言**: TypeScript
- **状态管理**: Redux / Zustand
- **导航**: React Navigation

## 功能模块

- 用户认证（邀请码注册、手机号登录）
- 训练计划查看
- 训练执行（动作演示、计时、反馈）
- 进度跟踪
- 复诊准备
- 患者教育
- 个人中心
